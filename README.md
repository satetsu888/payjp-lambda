# payjp-lambda

[pay.jp](https://pay.jp/)の課金APIを叩くAWS Lambda functionです。

pay.jpで用意されているもっとも簡単な実装方法である PAY.JP Checkout のバックエンドとして動作します。
これにより、サーバーサイドの処理を一切受け持つことなく、ページ内にタグを１つ貼るだけで課金システムが実装できます。
